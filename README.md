#Шаги установки: 

####1 - Подключить venv командой: 
` python -m venv venv`

####2 - Активировать его, командой:
- `cd venv`
- `cd Scripts`
- `activate`

####3 - Установить библиотеки командой:
`pip install -r requirements.txt`

####4 - В Консоле прейти в папку рут "mysite", командой:
`cd mysite`

####5 - Запустить сервер, командой:
`python manage.py`
