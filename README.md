#Шаги установки: 

####1 - Подключить venv командой: 
` python -m venv venv`

####2 - Установить библиотеки командой:
`pip install -r requirements.txt`

####3 - Активировать его, командой:
`cd Scripts activate`

####4 - В Консоле прейти в папку рут "mysite", командой:
`cd mysite`

####5 - Запустить сервер, командой:
`python manage.py`
